Restrict Language

Restrict is a small programming language exploring object-subject-verb composition, single-use values, and deterministic WebAssembly output.

Try the compiler Read the docs
pipeline.rl v0.0.1 surface 
fun increment: (value: Int32) -> Int32 = {
    value + 1
}

export fun score: () -> Int32 = {
    41 |> increment
}
typecheck ok host export: score() -> Int32

What is stable now

Release-facing surface

OSV-only calls

Values flow into verbs with `value |> function`, tuple calls, and unit calls.

Affine defaults

Bindings are checked as single-use unless the type is copyable or explicitly mutable.

Focused host ABI

v0.0.1 keeps host-visible exports scalar while composite values stay inside Restrict.
